Linux软件安装与卸载

2023-04-10 文档区 707 字

软件包分类

Linux 安装包主要有两种：
Ubuntu 和 Debian 使用的是 .deb 格式的安装包
red hat centos 使用的是 .rpm 格式的安装包

软件包管理器

Ubuntu 和 Debian 使用的管理器是 apt
red hat centos 使用的管理器是 yum
Ubuntu 现在还推荐使用一种新的管理器 snap

atp命令行

命令：apt update —获取最新软件包列表, 同步sources.list
命令：apt upgrate —更新当前系统所有安装的软件包和相关依赖
命令：apt install —下载并安装软件包
命令：apt remove —卸载软件包
命令：apt purge —卸载软件包，并删除配置文件
命令：apt list —列出包含条件的软件包
命令：apt show —显示软件包详情信息
命令：apt search —搜索应用程序

换源

vi /etc/apt/sources.list

dpkg包

dpkg本身是一个底层工具，而APT则是位于上层的工具

安装

命令：dpkg -i 软件包名 —安装Deb软件包

卸载

命令：dpkg -r 软件包名 —卸载Deb软件包
命令：dpkg -p 软件包名 —卸载Deb软件包，并删除配置信息

其他

命令：sudo apt --fix-broken install —自动完成依赖关系的处理
命令：dpkg -l 软件包名 —查询指定软件包
命令：dpkg -s 软件包名 —查询指定软件包的详细说明

Snap包

是针对Linux和物联网设计的，类似一个容器，完全独立与系统

安装

命令：snap install 软件包名 --安装Snap软件包
命令：snap refresh 软件包名 --手动更新软件Snap软件包

其他

命令：snap list —列出采用Snap包的应用软件
命令：snap find 软件包名 —搜索snap软件包
命令：snap info 软件报名 —查询指定软件包的详情信息

解压包

以下是一些常见的Linux解压命令：
命令：tar -xvf filename.tar —解压tar文件
命令：tar -xzvf filename.tar.gz —解压gzip格式的tar文件
命令：tar -xjvf filename.tar.bz2 ----解压bzip2格式的tar文件
命令：unzip filename.zip ----解压zip文件
命令：unrar x filename.rar ----解压rar文件

源代码安装

编译

命令：sudo make —使用命令 make 在当前文件夹进行编译

make 会依据 Makefile 文件的设置，将源代码编译成可执行的二进
制文件

安装

命令：sudo make install —安装makefile中设置，安装到指定文件夹

多数源码安装包会提供一个 configure 文件，用来自动检测操作环
境，并可生 成 Makefile 文件
在当前文件夹执行此命令 ./configure

卸载

命令：sudo make uninstall —卸载
命令：sudo make clean —卸载

checkinstall 命令

checkinstall 是一个能将源代码制作成安装包的程序，并自动安装

sudo apt install checkinstall
命令：sudo checkinstall —安装并生成deb安装包
