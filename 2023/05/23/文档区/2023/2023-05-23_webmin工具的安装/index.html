webmin工具的安装

2023-05-23 文档区 80 字
Linux

通过webmin工具远程管理配置服务器

安装配置面板 WebMin

使用deb包进行安装

下载路径：http://sourceforge.net/projects/webadmin/files/webmin/

sudo dpkg -i webmin_1.981_all.deb

处理依赖关系

sudo apt install -f

启动服务

开启端口10000

sudo ufw allow 10000

重启

reboot

登录

https：//服务器 ip：10000
