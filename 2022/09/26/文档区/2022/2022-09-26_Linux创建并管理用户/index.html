Linux创建并管理用户

2022-09-26 文档区 218 字
Linux

创建用户名和密码

sudo useradd -m <用户名>

-m 　自动建立用户目录。

sudo passwd <用户名> # 之后输入两次密码

切换用户

su <用户名>

如果是新装系统，切换到su root时候报错（su: Authentication failure），就输入sudo passwd root设置root的密码。

给予权限

sudo vim /etc/sudoers
在user privilege这一项进行修改

删除用户

sudo userdel -r <用户名>

连接远程用户

使用ssh连接工具，Windows自带的PowerShell也可以

ssh uerlink@120.20.200.200 # 输入你的用户名和远程服务器的ip

如果每次使用ip连接，又没有国内备案的域名。就可以修改window的host文件添加本地域名解析到ip。

退出用户登录

ctrl + d 或者 exit

Linux设置默认启动用户

在PowerShell中输入
debian config --default-user root
