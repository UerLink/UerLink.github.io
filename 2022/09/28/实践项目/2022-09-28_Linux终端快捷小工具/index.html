终端快捷小工具（Terminal-small-tools）

一些终端可以快捷使用的小工具，但个人水平有限(初学者),所以功能很少。

下载地址

链接：https://github.com/UerLink/Terminal-small-tools

当前工具列表

  • 终端快捷翻译

  • 终端快捷命令

安装与卸载方法

安装

在项目根目录中运行 install.sh 文件

./install.sh

卸载

在项目根目录中运行 uninstall.sh 文件

./uninstall.sh

运行失败解决

  1. 快捷翻译需要安装php环境
sudo apt-get install php
  1. 快捷命令添加自定义命令后运行失败,请尝试添加 sudo 运行
sudo run 编号
